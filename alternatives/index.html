VVO Iframe - Ostbahnhof

A simple iframe-based fallback using VVO Online departure monitors. Displays three stacked iframes filling the entire viewport.

Features: Full-screen display, responsive layout, minimal dependencies (jQuery CDN)

Drawbacks: 3 iframes that scale based on screen width, which means, it might not be usable out of the box. No rouite filtering.
